## Data:

- *user_id* — ID pengguna.
- *track* — judul trek lagu.
- *artist* — nama artis.
- *genre* — jenis lagu.
- *city* — kota tempat pengguna berada.
- *time* — lama waktu lagu tersebut dimainkan.
- *day* — hari dalam seminggu.

## Goal:

Membandingkan preferensi musik pengguna di kota Springfield dan Shelbyville.
